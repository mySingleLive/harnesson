v8 变更说明 — Claude Code 风格输入框

输入框结构

  • 圆角胶囊式输入框 (border-radius: 16px)
  • Focus 时紫色边框高亮
  • 上下文标签区 (@ 文件引用、Tools)
  • 多行输入 (自动扩展高度)

工具栏 (从左到右)

  • + 按钮 — 弹出菜单: 图片/文件引用/斜杠命令/Tools/MCP
  • 模型选择器 — 胶囊式, 点击切换模型
  • 发送按钮 — 紫色上箭头, Enter 发送
  • 底部快捷键提示

快捷操作

  • @ 引用文件/代码
  • / 斜杠命令 (Skills)
  • ⌘ V 粘贴图片
  • Enter 发送 / Shift+Enter 换行

与 v7 差异

  • 移除独立 model-selector 行, 集成到底部工具栏
  • 新增上下文标签区 (@ 文件引用)
  • + 按钮替代原来的独立图标按钮
  • 输入框整体更紧凑、更统一
