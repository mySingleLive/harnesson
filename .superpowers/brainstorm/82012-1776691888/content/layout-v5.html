Agent 状态指示器说明

Running

绿色呼吸灯
  • 绿色圆点
  • 呼吸灯动画（脉冲发光）
  • 表示 Agent 正在执行任务

Waiting

黄色圆点
  • 黄色半透明圆点
  • 静态无动画
  • 表示等待输入或排队

Error

红色感叹号
  • 红色圆形 + 感叹号图标
  • Agent 名称变红
  • 表示出错需要处理

Idle

灰色圆点
  • 灰色圆点
  • 静态无动画
  • 表示空闲待命
