# E2E 测试计划：Agent 导航卡片呼吸灯动画修改

## 测试简述
修改左侧 Agent 导航卡片的呼吸灯动画，将尺寸缩放效果改为明暗变化（opacity），并将颜色从橙色改为主题紫色（#8b5cf6）。通过浏览器验证修改后的视觉效果。

## 测试目标列表
- [ ] 目标1：呼吸灯动画为明暗变化而非尺寸变化（scale 不变，opacity 变化）
- [ ] 目标2：呼吸灯颜色为主题紫色（#8b5cf6）

## 测试方案
启动开发服务器 → 导航到首页 → 触发 Agent 进入 running 状态 → 截图验证呼吸灯动画效果。

## 测试用例列表

### 目标 1：呼吸灯动画为明暗变化

**测试用例 1.1：验证 Agent running 状态呼吸灯无尺寸变化**
- **测试数据**：无需特殊数据
- **Given**：启动开发服务器 → 导航 http://localhost:5173 → 选择项目 harnesson
- **When**：发送一条消息触发 Agent 运行 → 等待 Agent 进入 running 状态 → 观察左侧 Agent 卡片状态指示灯
- **Then**：状态指示灯应呈现明暗变化（opacity 波动）→ 指示灯尺寸不应有明显缩放 → 通过 eval 检查动画 keyframes 确认无 transform: scale

### 目标 2：呼吸灯颜色为主题紫色

**测试用例 2.1：验证呼吸灯颜色为主题紫色**
- **测试数据**：无需特殊数据
- **Given**：Agent 处于 running 状态（复用用例 1.1 的状态）
- **When**：截图观察左侧 Agent 卡片状态指示灯
- **Then**：指示灯颜色应为紫色（#8b5cf6）而非橙色 → 通过 eval 检查动画 keyframes 确认颜色值

## 验证方案
- 通过 eval 检查 CSS keyframes 定义确认动画属性（无 scale、有 opacity、颜色为 #8b5cf6）
- 通过浏览器截图肉眼确认视觉效果符合预期
